# QuillQuest 

QuillQuest is a simple book discovery web app built using HTML, CSS, and JavaScript.

## Features
- Search books using Google Books API
- Filter results
- Dynamic rendering (no frameworks)

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
- Google Books API

## How to Run
1. Clone the repo
2. Open index.html in browser
